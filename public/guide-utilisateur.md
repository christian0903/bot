# Guide Utilisateur — Back on Track

## Bienvenue

Back on Track est l'application de votre studio de fitness. Elle vous permet de consulter le planning des cours, réserver vos séances, gérer vos packs de crédits et suivre votre activité.

---

## Créer un compte

1. Cliquez sur **Connexion** dans le header
2. Sélectionnez l'onglet **Inscription**
3. Remplissez le formulaire :
   - **Nom d'affichage** : votre prénom ou pseudo (lettres, espaces et tirets uniquement)
   - **Email** : votre adresse email
   - **Mot de passe** : 8 caractères minimum
   - **Confirmation** : retapez le mot de passe
   - **Question de vérification** : répondez à la question mathématique simple
4. Cliquez sur **S'inscrire**
5. **Vérifiez votre boîte email** : vous recevrez un lien de confirmation. Votre compte ne sera actif qu'après avoir cliqué dessus.

> **Note** : si vous ne trouvez pas l'email, vérifiez vos spams.

## Se connecter

1. Cliquez sur **Connexion**
2. Entrez votre email et mot de passe
3. Cliquez sur **Se connecter**
4. Vous êtes redirigé vers votre tableau de bord

## Mot de passe oublié

1. Sur la page de connexion, cliquez sur l'onglet **Mot de passe oublié ?**
2. Entrez votre adresse email
3. Cliquez sur **Envoyer le lien**
4. Consultez votre email et cliquez sur le lien reçu
5. Choisissez un nouveau mot de passe (8 caractères minimum)

## Navigation

### Sur ordinateur
Le header contient les liens vers toutes les sections :
- **Accueil** — Page d'accueil avec annonces du studio
- **Planning** — Voir et réserver les cours
- **Mes réservations** — Historique de vos réservations
- **Mes packs** — Vos packs et crédits restants
- **Acheter un pack** — Catalogue des packs disponibles

### Sur mobile
Une **barre de navigation en bas** de l'écran donne un accès rapide aux 4 sections principales :
- 🏠 Accueil
- 📅 Planning
- 💳 Mes crédits
- 👤 Profil

---

## Tableau de bord

Après connexion, votre tableau de bord affiche :

- **Message de bienvenue** personnalisé
- **Accès rapide** : deux cartes cliquables vers le Planning et l'achat de packs
- **Mes crédits** : vos packs actifs avec une barre de progression colorée pour chaque pack
  - La couleur indique le type de crédit (bleu = semi-privé, orange = personal training)
  - Le nombre de jours restants avant expiration est affiché (en rouge si < 14 jours)
- **Prochains cours** : vos prochaines réservations avec la date, l'heure et le coach

## Le planning des cours

### Accéder au planning
Cliquez sur **Planning** dans la navigation ou sur la carte "Voir le planning" du tableau de bord.

### Naviguer dans le planning
- Utilisez les **flèches ◀ ▶** pour changer de semaine
- Cliquez sur la **date au centre** pour revenir à la semaine actuelle
- Basculez entre **vue liste** (recommandée sur mobile) et **vue semaine** avec les boutons en haut à droite

### Comprendre les cartes de cours
Chaque cours affiche :
- **L'heure** et la durée
- **Le nom du cours** (avec un point de couleur selon le type)
- **Le coach** (avec sa photo si disponible)
- **Les places restantes** (ex : 3/8)

### Réserver un cours
1. Trouvez le cours souhaité dans le planning
2. Cliquez sur le bouton **Réserver**
3. La réservation est immédiate — un crédit du type correspondant est automatiquement consommé
4. Le bouton change en ✅ **Réservé**

> **Pas de crédits ?** Un message vous indiquera que vous n'avez pas de crédits disponibles pour ce type de cours. Rendez-vous dans **Acheter un pack** pour en obtenir.

### Cours complet — Liste d'attente
Si un cours affiche **Complet** :
1. Cliquez sur **Liste d'attente**
2. Votre position dans la file est affichée (ex : "En attente (2)")
3. Si une place se libère, vous recevez une **notification** : "Place disponible !"
4. Vous avez alors **2 heures** pour confirmer en cliquant sur le bouton orange **Confirmer ma place**
5. Passé ce délai, la place est proposée au suivant

Pour quitter la liste d'attente, cliquez sur le **✕** à côté du badge.

### Événements spéciaux
Certains cours ont un **titre personnalisé** (ex : "Conférence Nutrition Sportive"). Ils apparaissent avec leur titre en gras et le type de cours en dessous.

## Mes réservations

Cette page liste toutes vos réservations, réparties en deux onglets :

### À venir
- Vos prochaines séances avec date, heure, nom du cours et coach
- Bouton **Annuler** pour annuler une réservation (le crédit n'est pas restitué automatiquement — contactez l'administration si besoin)

### Passées
- Historique de vos séances passées et annulées
- Chaque réservation affiche son statut : **Confirmée** ou **Annulée**

---

## Mes packs et crédits

### Consulter vos packs
La page **Mes packs** affiche tous vos packs :
- **Nom du pack** et type de crédit
- **Barre de progression** : crédits restants / crédits totaux
- **Date d'expiration** (en rouge si < 14 jours)
- **Prix payé**
- Les packs expirés ou épuisés apparaissent en grisé

### Acheter un pack
1. Allez dans **Acheter un pack**
2. Parcourez les packs disponibles pour votre catégorie de membre
3. Chaque carte affiche :
   - Le type de crédit
   - Le nombre de crédits inclus
   - La durée de validité
   - Le prix total et le prix par séance
4. Cliquez sur **Acheter**
5. Vous êtes redirigé vers la page de paiement sécurisée **Stripe**
6. Après paiement, vos crédits sont immédiatement activés et vous recevez une notification

> Le pack le plus populaire est mis en avant avec un badge **Populaire**.

## Profil

### Modifier votre profil
1. Cliquez sur votre **avatar** dans le header → **Profil**
2. Vous pouvez modifier :
   - Nom d'affichage
   - Prénom et nom
   - Téléphone
   - Catégorie de membre
   - Bio
3. Cliquez sur **Enregistrer**

### Profil public
Les autres membres peuvent voir votre profil public (nom d'affichage et bio uniquement).

---

## Notifications

- La **cloche** 🔔 dans le header affiche le nombre de notifications non lues
- Cliquez dessus pour accéder à la page des notifications
- Types de notifications :
  - ✅ Pack attribué ou acheté
  - 📅 Place disponible (liste d'attente)
  - ℹ️ Informations du studio
- Bouton **Tout marquer comme lu** pour nettoyer les notifications

---

## Langue et thème

### Changer la langue
Cliquez sur **FR** ou **EN** dans le header. La langue est détectée automatiquement au premier accès selon votre navigateur.

### Changer le thème
Cliquez sur l'icône ☀️/🌙 dans le header pour choisir parmi 4 thèmes :
- **Classique** — Fond clair, sobre
- **Sombre** — Fond noir, reposant pour les yeux
- **Vivid** — Fond clair avec couleurs vives
- **Vivid sombre** — Fond noir avec couleurs vives

Votre choix est sauvegardé automatiquement.

## Installer l'application (PWA)

Back on Track est une application web progressive. Vous pouvez l'installer sur votre appareil pour y accéder comme une app native :

- **iPhone / iPad** : ouvrez Safari → bouton Partager → **Sur l'écran d'accueil**
- **Android** : ouvrez Chrome → Menu ⋮ → **Installer l'application**
- **Ordinateur** : Chrome → icône d'installation dans la barre d'adresse

Une fois installée, l'application est accessible depuis votre écran d'accueil et fonctionne même avec une connexion limitée.
