# Guide Utilisateur - Back on Track

## Premiers pas

### Inscription
1. Cliquez sur **Connexion** dans le header
2. Sélectionnez l'onglet **Inscription**
3. Remplissez le formulaire : nom d'affichage, email, mot de passe
4. Répondez à la question de vérification
5. Cliquez sur **S'inscrire**
6. Vérifiez votre boîte email et cliquez sur le lien de confirmation

### Connexion
1. Cliquez sur **Connexion**
2. Entrez votre email et mot de passe
3. Cliquez sur **Se connecter**

### Mot de passe oublié
1. Sur la page de connexion, cliquez sur **Mot de passe oublié ?**
2. Entrez votre email
3. Cliquez sur **Envoyer le lien**
4. Vérifiez votre email et suivez le lien pour créer un nouveau mot de passe

## Navigation

- **Accueil** : Annonces du studio et accès rapide
- **Planning** : Voir tous les cours disponibles et réserver
- **Mes réservations** : Historique de vos réservations (à venir et passées)
- **Mes packs** : Vos packs achetés et crédits restants
- **Acheter un pack** : Catalogue des packs disponibles

## Réserver un cours

1. Allez dans **Planning**
2. Naviguez avec les flèches pour changer de semaine
3. Basculez entre vue semaine et vue liste avec les boutons en haut à droite
4. Cliquez sur **Réserver** sur le cours souhaité
5. Si vous avez des crédits disponibles du bon type, la réservation est confirmée
6. Si vous n'avez pas de crédits, vous serez invité à acheter un pack

## Acheter un pack

1. Allez dans **Acheter un pack**
2. Consultez les packs disponibles pour votre catégorie
3. Cliquez sur **Acheter** sur le pack choisi
4. Vous serez redirigé vers la page de paiement sécurisée Stripe
5. Après le paiement, vos crédits sont automatiquement activés

## Profil

1. Cliquez sur votre avatar dans le header → **Profil**
2. Modifiez votre nom, prénom, téléphone, bio et catégorie de membre
3. Cliquez sur **Enregistrer**

## Thèmes et langue

- **Langue** : Cliquez sur **FR** ou **EN** dans le header pour changer de langue
- **Thème** : Cliquez sur l'icône soleil/lune dans le header pour choisir parmi 4 thèmes :
  - Classique (clair)
  - Sombre
  - Vivid (coloré)
  - Vivid sombre

## Notifications

- La cloche dans le header affiche le nombre de notifications non lues
- Cliquez dessus pour voir toutes vos notifications
- Vous pouvez marquer toutes les notifications comme lues

## Application mobile (PWA)

Back on Track est une Progressive Web App. Pour l'installer :
- **iPhone/iPad** : Ouvrez Safari → Bouton Partager → "Sur l'écran d'accueil"
- **Android** : Chrome → Menu (⋮) → "Installer l'application"
- **Desktop** : Chrome → Barre d'adresse → Icône d'installation
